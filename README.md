# Sky Hopper 🐦

This is a Flappy Bird–style game built with Python and Pygame. Challenge yourself to guide the bird through endless pipes in this addictive arcade adventure.

## Features

- Classic Flappy Bird mechanics with smooth controls.
- Randomly generated pipes with varying gaps.
- Collision detection for pipes and ground.
- Scoring system that tracks successful passes.
- Start screen and game-over screen with final score display.
- Lightweight and easy to run locally.

## How to Play

1. Press the **Spacebar** to flap and keep the bird airborne.
2. Avoid hitting pipes or the ground.
3. Earn points by successfully passing through gaps.
4. Keep playing to beat your high score!
